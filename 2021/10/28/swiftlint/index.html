遇到的问题

1、for in 的方法使用，以及数组的enumerated()方法使用，比如：

之前的状态

for car in cars { if car == "" { // } }

改后的状态

for car in cars where car == "" { // }

之前的状态

for (index, _) in cell.dataSource.enumerated()

这样的就应该直接用

for index in 0 ..< cell.dataSource.count

2、判断的使用，比如：

之前的状态

if let _ = car

guard let _ = car

之后的状态

if car != nil

guard car != nil

3、Switch内的case，不需要在case里面写break
4、工程内有比较多的try!，目前做的是警告处理，尽量防止出问题，还是if let * = try?
